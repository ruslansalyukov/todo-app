# 📝 To-Do List App

Простое и удобное приложение для управления задачами (To-Do List) ⚡

## ✨ О проекте

To-Do List — это мини-приложение, разработанное на JavaScript.  
Оно позволяет удобно добавлять, выполнять и удалять задачи, а также сохраняет данные между сессиями с помощью локального хранилища браузера (localStorage) 💾

## 🎯 Цель проекта

Проект создан для практики JavaScript и закрепления базовых навыков фронтенд-разработки.

Также он может использоваться как простой персональный инструмент для:
- 📌 планирования задач  
- ✅ отслеживания выполненных дел  
- 🧠 организации повседневных дел  

## 🛠️ Технологии

- 🧱 HTML — структура приложения  
- 🎨 SCSS — стилизация интерфейса  
- ⚙️ JavaScript — логика приложения 

## ⚙️ Функционал

- ➕ Добавление новых задач  
- ✔️ Отметка задач как выполненных  
- 🗑️ Удаление задач  
- 💾 Сохранение данных в localStorage  
- 🔄 Автоматическое восстановление списка при перезагрузке
